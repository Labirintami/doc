## Attaching event listeners

You can attach event listeners with the <strong>popup.events.on()</strong> method:

~~~js
popup.events.on("Click", function(e){
    console.log("The popup was clicked");
});
~~~

:::note
The names of events are case-insensitive.1
:::

## Detaching event listeners

To detach events, use <strong>popup.events.detach()</strong>:

~~~js
popup.events.on("Click", function(e){
    console.log("The popup was clicked");
});

popup.events.detach("Click");
~~~

## Calling events

To call events, use <strong>popup.events.fire()</strong>:

~~~js
popup.events.fire("name",args);
// where args is an array of arguments
~~~
