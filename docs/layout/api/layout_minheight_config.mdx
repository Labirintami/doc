<Short>
 the minimal height to be set for a cell






</Short>



<Type>
 number,string


</Type>


~~~js
 
var layout = new dhx.Layout("layout_container", {
    cols: [
      { header: "Cell header", minHeight: "40px"}
    ]
});


~~~
	api_config

<Descr>
 
In case both <strong>height</strong> and <strong>minHeight</strong> properties are set in the configuration of a cell, the <strong>minHeight</strong> option is a priority.


</Descr>
