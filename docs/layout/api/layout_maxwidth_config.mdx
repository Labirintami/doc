<Short>
 the maximal width to be set for a cell






</Short>



<Type>
 number,string


</Type>


~~~js
 
var layout = new dhx.Layout("layout_container", {
    cols: [
      { header: "Cell header", maxWidth: "700px"}
    ]
});


~~~
	api_config

<Descr>
 
In case both <strong>width</strong> and <strong>maxWidth</strong> properties are set in the configuration of a cell, the <strong>maxWidth</strong> option is a priority.


</Descr>
