    ]
});

	api_config

<Descr>
 
Starting from v7.0, the height of a cell can be automatically adjusted to the height of its content. For that, set the <strong>height</strong> configuration option to *"content"*:

~~~js
var layout = new dhx.Layout("layout_container", {
    cols: [
      { header: "Cell header", height: "content" } /*!*/
    ]
});
~~~


</Descr>
