<Short>
 the minimal width to be set for a cell






</Short>



<Type>
 number,string


</Type>


~~~js
 
var layout = new dhx.Layout("layout_container", {
    cols: [
      { header: "Cell header", minWidth: "400px"}
    ]
});


~~~
	api_config

<Descr>
 
In case both <strong>width</strong> and <strong>minWidth</strong> properties are set in the configuration of a cell, the <strong>minWidth</strong> option is a priority.


</Descr>
