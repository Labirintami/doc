<Short>
 
sets the width of a cell





</Short>



<Type>
 string|number


</Type>


~~~js
 
var layout = new dhx.Layout("layout_container", {
    cols: [
      { header: "Cell header", width: "300px" }
    ]
});


~~~
	api_config

<Descr>
 
Starting from v7.0, the width of a cell can be automatically adjusted to the width of its content. For that, set the <strong>width</strong> configuration option to *"content"*:

~~~js
var layout = new dhx.Layout("layout_container", {
    cols: [
      { header: "Cell header", width: "content" } /*!*/
    ]
});
~~~





</Descr>
