## Attaching event listeners

You can attach event listeners with the <strong>pagination.events.on()</strong> method:

~~~js
pagination.events.on("Change", function(index, previousIndex){
    console.log("The currently active page", index);
});
~~~

:::note
The names of events are case-insensitive.1
:::

## Detaching event listeners

To detach events, use <strong>pagination.events.detach()</strong>:

~~~js
pagination.events.on("Change", function(index, previousIndex){
    console.log("The currently active page", index);
});

pagination.events.detach("Change");
~~~

## Calling events

To call events, use <strong>pagination.events.fire()</strong>:

~~~js
pagination.events.fire("name",args);
// where args is an array of arguments
~~~
