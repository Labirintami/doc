<Short>
 
enables selection of multiple DataView items, sets the mode of multiselection





</Short>



<Type>
 boolean,string


</Type>


 true, false, "click", "ctrlClick" 

~~~js
 
var dataview = new dhx.DataView("list", {multiselection:true});


~~~
	api_config

<Descr>
 
Setting the <strong>multiselection</strong> property to *true* presupposes selection of multiple items by using Ctrl key.




</Descr>
