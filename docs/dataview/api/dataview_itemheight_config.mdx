<Short>
 
sets the height of an item



</Short>



<Type>
 number|string


</Type>


~~~js
 
// sets the height of an item as a number
var dataview = new dhx.DataView("dataview", { 
    itemHeight:30
});

// sets the height of an item as a string value
var dataview = new dhx.DataView("dataview", { 
    itemHeight:"30px"
});


~~~
	api_config

<Descr>
 
The usage of the *CSS calc() function* within the <strong>itemHeight</strong> property is not possible.


</Descr>
