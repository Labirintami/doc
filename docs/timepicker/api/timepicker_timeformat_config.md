timeFormat
=============

@short: 
defines what clock format is activated: the 12-hour or 24-hour one


@default:
24


@type: number

@example: 
var timepicker = new dhx.Timepicker("timepicker", {
	timeFormat: 12
});


@template:	api_config
@descr:
