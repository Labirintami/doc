 https://snippet.dhtmlx.com/5ccptwy7	Timepicker. Timepicker Events

 

- added in v6.4
- deprecated in v7.0

instead of it, you can use the timepicker/api/timepicker_afterapply_event.md and timepicker/api/timepicker_beforeapply_event.md events:

~~~js
timepicker.events.on("AfterApply", function(value){
	console.log(value);
});

timepicker.events.on("BeforeApply", function(value){
	console.log(value);
    return false;
});
~~~
