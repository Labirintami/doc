## Attaching event listeners

You can attach event listeners with the <strong>timepicker.events.on()</strong> method:

~~~js
timepicker.events.on("Change", function(value){
    console.log("The value of a timepicker has changed to "+ value);
});
~~~

:::note
The names of events are case-insensitive.1
:::

## Detaching event listeners

To detach events, use <strong>timepicker.events.detach()</strong>:

~~~js
timepicker.events.on("Change", function(value){
    console.log("The value of a timepicker has changed to "+ value);
});
timepicker.events.detach("Change");
~~~

## Calling events

To call events, use <strong>timepicker.events.fire()</strong>:

~~~js
timepicker.events.fire("name",args);
// where args is an array of arguments
~~~

The list of events is available in the [API section](/timepicker/api/refs/timepicker_events).
