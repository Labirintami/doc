<Short>
 fires before a row is shown on a page


</Short>


 
- row   object  an object with a row configuration



- result	boolean		false - to prevent a row from being shown, otherwise true

~~~js

treegrid.events.on("BeforeRowShow", function(row){
    // your logic here
    return false;
});


~~~
