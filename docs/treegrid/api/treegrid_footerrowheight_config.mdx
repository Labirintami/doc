<Short>
 
sets the height of rows in the footer



</Short>


<strong>Default value: </strong> 
40



<Type>
 number


</Type>


~~~js
 
var grid = new dhx.Grid("grid_container", {
	columns: [// columns config],
	footerRowHeight: 50
});


~~~
	api_config

<Descr>
 
The height of the footer is calculated as a sum of all row heights in it.


</Descr>
