<Short>
 adjusts the width of a column to make all its content visible



</Short>



- id		string,number		the id of a column
- adjust	string,boolean		optional, the mode of adjusting a column ("header", "data", true)


~~~js

treegrid.adjustColumnWidth("b", "header");


~~~
 api_method

<Descr>

If no value is set to the *adjust* parameter, the <strong>adjustColumnWidth</strong> method sets it to *true*.


</Descr>
