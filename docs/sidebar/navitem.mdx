This is a default control for navigation-related components, can contain any other controls as nested items. 

<img style="margin: 0px 0px 0px 20px; display: block;" src="sidebar/navitem.png" alt=""/>

Adding NavItem
----------------

A navItem can be easily added to a sidebar with the help of the <strong>add()</strong> method of Tree Collection:

~~~js
sidebar.data.add({
    type:"navItem", value:"My NavItem"
});
~~~

## Adding an icon

A navItem can have an icon which is set through the corresponding option <strong>icon</strong>:

~~~js
{
    type:"navItem", value:"Some",
    icon:"dxi dxi-check"
}
~~~

## Adding a badge with a number

You can add a number badge to the navItem to display information like the number of new messages. The badge is set via the <strong>count</strong> property:

~~~js
{
    type:"navItem", value:"Some",
    icon:"dxi dxi-check",
    count:10
}
~~~

## Adding HTML content

You can add any custom HTML content to a navItem with the help of the <strong>html</strong> property:

~~~js
{
    type: "navItem",
    html: "<img class='item-avatar' src='../avatars/Avatar_07.jpg'/>"
}
~~~

## Showing/hiding a navItem

To hide/show a navItem, you should pass the ID of the navItem to the <strong>hide() / show()</strong> Sidebar methods:

~~~js
sidebar.show(id);
sidebar.hide(id);
~~~

## Enabling/disabling a navItem 

Any navItem in the sidebar can be enabled/disabled:

~~~js
sidebar.enable(id);
sidebar.disable(id);
~~~

## Setting tooltip

You can add a tooltip to a navItem:

~~~js
{
    type:"navItem", 
    value:"Click", 
    tooltip:"Click me and find out why"  /*!*/
}
~~~
