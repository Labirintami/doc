## Adding Input

The following example shows how an Input control can be easily added to a sidebar with the help of the <strong>add()</strong> method of Tree Collection:

~~~js
sidebar.data.add({
	type:"input",
	value:"",
	placeholder:"Type to search"
});
~~~


### Attributes

You can provide the following <strong>attributes</strong> in the configuration object of an input field:

- <strong>type</strong> - (*string*) required, the type of a control, set it to "input". If not specified - the ["navItem"](/sidebar/navitem) type is applied by default.
- <strong>id</strong> - (*string*) the id of a control, auto-generated if not set
- <strong>icon</strong> - (*string*) optional, the name of an [icon](/sidebar/customization#icons) from the used icon font
- <strong>placeholder</strong> - (*string*) optional, a tip for the input 
- <strong>width</strong> - (*number*) optional, the width of the input field
- <strong>label</strong> - (*string*) optional, a text label for the Input control
- <strong>value</strong> - (*string*) optional, the initial value of the field
- <strong>tooltip</strong> - (*string*) a tooltip for an input
- <strong>inputType</strong> - (*string*) sets the type of an input, use the "password" value to specify a field for entering a password


## Showing/hiding input 

You can show or hide Input with the methods of Sidebar:

~~~js
sidebar.show(id);
sidebar.hide(id);
~~~

## Enabling/disabling input

You can also disable and enable inputs:

~~~js
sidebar.enable(id);
sidebar.disable(id);
~~~

## Setting tooltip

You can create a tooltip for an input:

~~~js
{
	type: "input",
	value: "",
	tooltip:"Type to search"
}
~~~


## Setting/getting value

You can use the <strong>setState()</strong> and <strong>getState()</strong> methods of Sidebar to change and access the value of Input.

To fill in Input with some text, call <strong>setState()</strong> with a key-value pair as a parameter, where the key is the ID of the input and the value is the text:

~~~js
{
	type: "input",
	id:"search",
	value: ""
}
...
sidebar.setState({search:"Summer"});
~~~

To get the text that is currently inside the Input control, call <strong>getState()</strong>:

~~~js
sidebar.getState(); //-> {search:"Summer"}
~~~
