<Short>
 
makes Combo Box readonly



</Short>


<strong>Default value: </strong> 
false



<Type>
 boolean


</Type>


~~~js
 
var combo = new dhx.Combobox("combo_container", { 
    readOnly:true
});


~~~
	api_config

<Descr>
 

:::note
Note, that in v7.0, the <strong>readonly</strong> property have been deprecated and renamed to <strong>readOnly</strong>. 1
:::

In this mode it is only possible to select options from the list, without entering words in the input.
