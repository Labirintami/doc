<Short>
 
defines whether the Select All button should be shown





</Short>



<Type>
 boolean


</Type>


~~~js
 
var combo = new dhx.Combobox("combo_container", {
    // enables multiple selection of options
    multiselection:true,
    // shows the Select All button
    selectAllButton: true
});


~~~
	api_config

<Descr>
 
Note that to select several options you need to enable the <strong>multiselection</strong> option.


</Descr>
