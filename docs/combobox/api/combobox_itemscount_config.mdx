<Short>
 shows the total number of selected options



</Short>



<Type>
 boolean,function



</Type>


~~~js
 
var combo = new dhx.Combobox("combo_container", {
    multiselection: true,
    itemsCount: function (num) {
        return num + (num === 1 ? " item" : " items") + " selected";
    }
});


~~~
	api_config

<Descr>
 
As a function takes one parameter:

- <strong>count</strong> - (*number*) the number of options

and should return a template for displaying a result.

Note that to select several options you need to enable the <strong>multiselection</strong> option.
 

</Descr>
