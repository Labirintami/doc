autoEmptyRow
=============

@short: 
adds an empty row after the last filled row in the Grid


@default:true


@type:boolean

@example: 
var grid = new dhx.Grid("grid_container", {
	columns: [// columns config],
	autoEmptyRow:true,  /*!*/
	data: dataset
});


@template:	api_config
