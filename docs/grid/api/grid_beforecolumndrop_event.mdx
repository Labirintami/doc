<Short>
 fires before the user has finished dragging and released the mouse button over a target column
	

</Short>



- data		object		data object
- e		    Event		a native event object


- result	boolean		false - to block the default action, otherwise true


~~~js

grid.events.on("BeforeColumnDrop", function(data, events){
	// your logic here
    return false;
});


~~~
	api_event

<Descr>

:::danger Pro Note
This functionality requires PRO version of the dhtmlxGrid (or DHTMLX suite) package.
:::

The data object contains the following parameters:

- <strong>start</strong> - (*string*) the id of a column, from which the dragging process has started
- <strong>source</strong> - (*string[]*) an array with ids of dragged columns
- <strong>target</strong> - (*string*) the id of a potential target column


</Descr>
