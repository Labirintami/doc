<Short>
 
defines whether sorting on clicking headers of columns is enabled



</Short>


<strong>Default value: </strong> 
true



<Type>
 boolean


</Type>


~~~js
 
var grid = new dhx.Grid("grid_container", {
	columns: [// columns config],
	headerSort:false,  /*!*/
	data: dataset
});


~~~
	api_config

<Descr>
 



</Descr>


 instead of it, you can use the grid/api/grid_sortable_config.md property:

~~~js
var grid = new dhx.Grid("grid_container", {
	columns: [// columns config],
	sortable:false,  /*!*/
	data: dataset
});
~~~
