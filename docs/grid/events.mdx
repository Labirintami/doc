## Attaching event listeners

You can attach event listeners with the <strong>grid.events.on()</strong> method:

~~~js
grid.events.on("Scroll", function({top,left}){
    console.log("The grid is scrolled to "+top,left);
});
~~~

:::note
The names of events are case-insensitive.1
:::

## Detaching event listeners

To detach events, use <strong>grid.events.detach()</strong>:

~~~js
grid.events.on("Scroll", function({top,left}){
    console.log("The grid is scrolled to "+top,left);
});

grid.events.detach("Scroll");
~~~

## Calling events

To call events, use <strong>grid.events.fire()</strong>:

~~~js
grid.events.fire("name",args);
// where args is an array of arguments
~~~
