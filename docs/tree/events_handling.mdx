## Attaching event listeners

You can attach event listeners with the <strong>tree.events.on()</strong> method:

~~~js
tree.events.on("ItemClick", function(id, e){
    console.log("The item with the id "+ id +" was clicked.");
});
~~~

:::note
The names of events are case-insensitive.1
:::

## Detaching event listeners

To detach events, use <strong>tree.events.detach()</strong>:

~~~js
tree.events.on("ItemClick", function(id, e){
    console.log("The item with the id "+ id +" was clicked.");
});

tree.events.detach("ItemClick");
~~~

## Calling events

To call events, use <strong>tree.events.fire()</strong>:

~~~js
tree.events.fire("name",args);
// where args is an array of arguments
~~~
