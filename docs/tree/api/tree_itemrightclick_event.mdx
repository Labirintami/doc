<Short>

fires on right-clicking a tree item


</Short>



- id	string		the id of an item
- e		Event		a native event object

~~~js

tree.events.on("itemRightClick", function(id, e){
    console.log("The item with the id "+ id +" was right-clicked.");
});


~~~
