<Short>
 
allows adding custom icons for tree items





</Short>



<Type>
 object


</Type>


~~~js
 
var tree = new dhx.Tree("tree_container", {
	icon: {
		folder: "fas fa-book",
		openFolder: "fas fa-book-open",
		file: "fas fa-file"
	}
});


~~~
	api_config

<Descr>
 
There are three types of [icons](/tree/setting_tree_appearance#changingicons) that can be customized:

- file
- folder
- openFolder

You should specify the name of the necessary icon type in the <strong>icon</strong> object and set the name of an icon from the used icon pack as its value.


</Descr>
