<Short>
 edits a Tree item



</Short>



- id		string		the id of an item
- config	object		optional, the config object of an editor



~~~js

tree.editItem(id);


~~~
 api_method

<Descr>

			</td>
		</tr>
		<tr>
			<td class="webixdoc_links0"><b>options</b></td>
