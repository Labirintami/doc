<Short>
 returns the index of an item in focus



</Short>






- index		number		the index of an item in focus


~~~js

var index = list.getFocusIndex(); // -> 6


~~~
 api_method

<Descr>



</Descr>


 To get the index of a List item in focus you can get the id of an item using the [getFocus()](/list/api/list_getfocus_method) method and pass this id as a parameter of the data_collection/api/getindex.md method of Data Collection:
~~~js
var id = list.getFocus();
var index = list.data.getIndex(id);
~~~

But we recommend you to use the [getFocus()](/list/api/list_getfocus_method) method instead:

~~~js
list.getFocus();
~~~
