@short: 
defines that an item is copied to a target during drag-n-drop




@type: boolean

@example: 
var source = new dhx.List("list-source", {dragMode: "source", dragCopy: true});
var target = new dhx.List("list-target", {dragMode: "target", dragCopy: true});
