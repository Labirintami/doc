## Attaching event listeners

The user can add any user-defined handler to any of the available events. To do this, the user can use the <strong>list.events.on()</strong> method with the following parameters:

<table class="webixdoc_links">
	<tbody>
        <tr>
			<td class="webixdoc_links0"><b>evName</b></td>
			<td>name of the event</td>
		</tr>
        <tr>
			<td class="webixdoc_links0"><b>evHandler</b></td>
			<td>user-defined event handler</td>
		</tr>
    </tbody>
</table>
<br/>
  
~~~js
list.events.on("Click", function(id, e){
    console.log("The item with the id "+ list.selection.getId(id) +" was clicked.");
});
~~~

Several handlers can be attached to one and the same event, and all of them will be executed.

:::note
The names of the events are case-insensitive.

## Detaching event listeners

There is a simple way of removing an event handler via the <strong>list.events.detach()</strong> method:

~~~js
list.events.on("Click", function(id, e){
    console.log("The item with the id "+ list.selection.getId(id) +" was clicked.");
});

list.events.detach("Click");
~~~

## Calling events

To call events, use <strong>list.events.fire()</strong>:

~~~js
list.events.fire("name",args);
// where args is an array of arguments
~~~
