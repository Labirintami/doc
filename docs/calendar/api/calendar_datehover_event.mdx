<Short>

fires when the mouse pointer is over a date


</Short>



- date  	Date    	the date under the pointer
- e 		Event		a native event object

~~~js

calendar.events.on("DateHover", function(date, e) {
    dhx.tooltip(date, {node: e.target, position: "bottom"});
});


~~~
