DHTMLX AwaitRedraw is a helper that allows implementing specified actions for a component as soon as possible after it is redrawn. 

The <strong>dhx.awaitRedraw</strong> helper returns a promise:

- <strong>promise</strong> - (*function*) a function to be executed

~~~js
dhx.awaitRedraw().then(function() {
    // your code here
})
~~~
