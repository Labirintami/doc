## Adding Text Control

A Text Control can be added to the ribbon with the help of the <strong>add()</strong> method of TreeCollection:

~~~js
myribbon.data.add({
    type: "text",
	value: "APP NAME"
});
~~~

## Attributes

You can provide the following attributes in the configuration object of Text:

- <strong>id</strong> (string|number) - optional, the ID of the control;
- <strong>type</strong> (string) - always "text";
- <strong>value</strong> (string) - optional, the text;
- <strong>$hidden</strong> (boolean) - optional, hides the control;
- <strong>css</strong> (string) - optional, adds a custom CSS class.

## Showing/Hiding Text

Text can be hidden and shown:

~~~js
myRibbon.show(id);
myRibbon.hide(id);
~~~
