## Adding menuItem

You can add a menuItem to the ribbon with the help of the <strong>add()</strong> method of the <strong>data</strong> module:

~~~js
myRibbon.data.add({
	type:"menuItem", value:"Menu", childs:[
		{ type:"menuItem", value:"Option 1" },
		{ type:"menuItem", value:"Option 2" },
		{ type:"menuOption", value:"Option Infinite" }
	]
});
~~~

## Attributes

You can provide the following attributes in the configuration object of a menuItem:

- <strong>id</strong> (string|number) - optional, the ID of the control;
- <strong>type</strong> (string) -  always "menuItem";
- <strong>value</strong> (string) - optional, the text label;
- <strong>icon</strong> (string) - the icon;
- <strong>hotkey</strong> (string) - optional, adds a hotkey to a menu item that has no childs (otherwise adds it, but it doesn't open the menu item childs, just "clicks" the menu item itself);
- <strong>$disabled</strong> (boolean) - optional, disables the menu option;
- <strong>css</strong> (string) - optional, adds a custom CSS class;
- <strong>childs</strong> (array of objects) - adds submenu options that can have all the above properties.

You can add [separators](/ribbon/separator) between menu options.

## Changing the width of popups with menu options

You can set the width of dropdown menus with the <strong>popupWidth</strong> property of the ribbon configuration:

~~~js
var myribbon = new dhx.ribbon("ribbon",{
    popupWidth:100
});
~~~

## Showing/Hiding menuItem

A menu item can be hidden and shown again with the methods of ribbon:

~~~js
ribbon.show(id);
ribbon.hide(id);
~~~

## Enabling/Disabling menuItem

Menu items can be enabled and disabled:

~~~js
ribbon.enable(id);
ribbon.disable(id);
~~~
