## Attaching event listeners

You can attach event listeners with the <strong>slider.events.on()</strong> method:

~~~js
slider.events.on("Change", function(newValue, oldValue, isRange){
    console.log("The value of a slider has changed to "+slider.getValue());
});
~~~



:::note
The names of events are case-insensitive.1
:::

## Detaching event listeners

To detach events, use <strong>slider.events.detach()</strong>:

~~~js
slider.events.on("Change", function(newValue, oldValue, isRange) {
    console.log("The value of a slider has changed to "+slider.getValue());
});

slider.events.detach("Change");
~~~

## Calling events

To call events, use <strong>slider.events.fire()</strong>:

~~~js
slider.events.fire("name",args);
// where args is an array of arguments
~~~
