<Short>
 
enables prompt messages with ticks values on hovering over the slider thumb



</Short>


<strong>Default value: </strong> 
true



<Type>
 boolean


</Type>


~~~js
 
var slider = new dhx.Slider("slider_container", { 
    thumbLabel:false
});


~~~
	api_config

<Descr>
 



</Descr>


 instead of it, you can use the slider/api/slider_tooltip_config.md property:

~~~js
var slider = new dhx.Slider("slider_container", { 
    tooltip:false
});
~~~
