<Short>

fires after a window is shown


</Short>



- position  object      an object with coordinates of the window's position

~~~js

dhxWindow.events.on("AfterShow", function(position){
    console.log("A window is shown");
});


~~~
 api_event

<Descr>

The <strong>position</strong> parameter contains two attributes:

- <strong>left</strong> - (*number*)	the left coordinate of the window's position (in pixels)
- <strong>top</strong> - (*number*)	the top coordinate of the window's position (in pixels)


</Descr>
