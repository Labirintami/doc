@short: 
defines whether a window can be closed


@default:
false


@type: boolean

@example: 
var dhxWindow = new dhx.Window({
    closable:true
});
dhxWindow.show();


@template:	api_config
@descr: 
A closable window gets the Close icon in the right top corner.
