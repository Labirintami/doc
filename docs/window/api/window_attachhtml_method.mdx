<Short>
 adds an HTML content into a dhtmlxWindow



</Short>



- html		string		an HTML content to be added into a window



~~~js

var dhxWindow = new dhx.Window({title: "Window"});
var html = "<h1>Header</h1><p>paragraph</p>";
dhxWindow.attachHTML(html);
dhxWindow.show();


~~~
