<Short>

fires on resizing a window


</Short>



- state     		object    an object with the new size and (or) position of a window
- oldState   	object    an object with the previous size and (or) position of a window
- side  object  an object that specifies the direction of resizing


~~~js

dhxWindow.events.on("Resize", function(state, oldState, side) {
    console.log("The window is resized" + state.left, state.top, 
                state.width, state.height)
});


~~~
 api_event

<Descr>

The <strong>state/oldState</strong> object contains two attributes:

- <strong>size</strong> - (*object*) optional, an object with the size of a window. It contains two options:
    - <strong>width</strong> - (*number*) the width of the window
    - <strong>height</strong> - (*number*) the height of the window
- <strong>position</strong> - (*object*)  optional, an object with the position of the window  
    - <strong>left</strong> - (*number*)	the left coordinate of the window's position (in pixels)
    - <strong>top</strong> - (*number*)	the top coordinate of the window's position (in pixels)

The <strong>side</strong> object can contain the following directions:

- <strong>left</strong> - (*boolean*) optional, the left direction
- <strong>right</strong> - (*boolean*) optional, the right direction
- <strong>top</strong> - (*boolean*) optional, the top direction
- <strong>bottom</strong> - (*boolean*) optional, the bottom direction


</Descr>
