## Attaching event listeners

You can attach event listeners with the <strong>dhxWindow.events.on()</strong> method:

~~~js
dhxWindow.events.on("Move", function(left,top){
    console.log("The window moved to "+left,top)
});
~~~

:::note
The names of events are case-insensitive.1
:::

## Detaching event listeners

To detach events, use <strong>dhxWindow.events.detach()</strong>:

~~~js
var onmove = dhxWindow.events.on("Move", function(left,top){
    console.log("The window moved to "+left,top)
});

dhxWindow.events.detach(onmove);
~~~

## Calling events

To call events, use <strong>dhxWindow.events.fire()</strong>:

~~~js
dhxWindow.events.fire("name",args);
// where args is an array of arguments
~~~
