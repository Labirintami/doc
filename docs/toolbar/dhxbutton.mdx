Adding buttons
-----------------

A DHX button can be easily added to a toolbar with the help of the <strong>add()</strong> method of DataCollection:

~~~js
myToolbar.data.add({
    type:"dhxButton", value:"Best button"
});
~~~


Attributes
----------------

The <strong>dhxButton</strong> object has the following properties:

- <strong>type</strong> - (*string*) required, the type of a control, set it to "dhxButton";
- <strong>id</strong> - (*string*) the id of a control, auto-generated if not set;
- <strong>icon</strong> - (*string*)	optional, an icon for the DHXButton;
- <strong>hotkey</strong> - (*string*) optional, the name of the hot key for the DHXButton;
- <strong>value</strong> - (*string*) optional, a value for the DHXButton;
- <strong>tooltip</strong> - (*string*) optional, a tooltip for the DHXButton.
