## Adding Title control

A Title control can be added to a toolbar with the help of the <strong>add()</strong> method of [Tree Collection](/tree_collection/api/refs/treecollection):

~~~js
toolbar.data.add({
    type: "title",
	value: "APP NAME"
});
~~~

## Showing/hiding title

Title can be hidden and shown with the help of the [show()](/toolbar/api/toolbar_show_method)/[hide()](/toolbar/api/toolbar_hide_method)  methods. Pass the id of a Title control as a parameter.

~~~js
toolbar.show(id);
toolbar.hide(id);
~~~

## Adding a tooltip

You can add a tooltip to a Title control:

~~~js
{
    type:"title",
    value:"2K",
    tooltip:"Your subscribers"
}
~~~
