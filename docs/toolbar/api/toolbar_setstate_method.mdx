<Short>
 sets values/states of controls



</Short>



- state 		object			a hash of <i>key:value</i> pairs, where the key is the ID of a control and the value is a new value/state of the control



~~~js

toolbar.setState({
    "search": "Hello",
    "edit": true,
    "align": "right"
});


~~~
