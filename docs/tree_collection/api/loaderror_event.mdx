<Short>
 fires when loading of data fails
	

</Short>



- error			string,object		info about the error


~~~js

data.events.on("LoadError", function(error){
	// your code here
});


~~~
	api_event

<Descr>

The <strong>error</strong> parameter can be either a string with the text of an error, or an object with two properties:

- status - the code of an error
- text - the text of an error



	

</Descr>
