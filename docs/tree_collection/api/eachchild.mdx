<Short>
iterates through child items of a control


</Short>



- id 		string		 the ID of a control
- cb        function     the function that will be applied to every child of the item  




~~~js

toolbar.data.eachChild("menu_1", item => {
    // disable items
    item.$disabled = true;
});
toolbar.paint();

~~~
