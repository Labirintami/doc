<Short>
 returns the id of a selected item
	

</Short>




- id	string|string[]	the id of a selected item

~~~js

var id = tree.selection.getId(); // -> "2"


~~~
	api_method

<Descr>

Starting from v7.0, the method can also return an array with ids of selected items if the <strong>multiselection</strong> property of List or DataView is enabled.


</Descr>
