closeButtons
=============

@short: 
adds close buttons for tabs


@default:
false


@type: boolean

@example: 
var tabbar = new dhx.Tabbar("tabbar_container", {
    closeButtons:true
});


@deprecated: instead of it, you can use the tabbar/api/tabbar_closable_config.md property:

~~~js
var tabbar = new dhx.Tabbar("tabbar_container", {
    closable:true
});
~~~

@template:	api_config
@descr:
