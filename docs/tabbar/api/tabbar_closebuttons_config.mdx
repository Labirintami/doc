<Short>
 
adds close buttons for tabs



</Short>


<strong>Default value: </strong> 
false



<Type>
 boolean


</Type>


~~~js
 
var tabbar = new dhx.Tabbar("tabbar_container", {
    closeButtons:true
});


~~~
 instead of it, you can use the tabbar/api/tabbar_closable_config.md property:

~~~js
var tabbar = new dhx.Tabbar("tabbar_container", {
    closable:true
});
~~~
