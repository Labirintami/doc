<Short>
 adds a new tab into a tabbar



</Short>



- view 		object		an object of a tab 
- index 	number		the position to add a tab into



~~~js

tabbar.addTab({tab: "tab-" + nextTab++, css:"panel flex"}, 0);


~~~
