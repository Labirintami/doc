<Short>
 removes a tab from a tabbar



</Short>



- id 		string 			the id of a tab to remove



~~~js

var id = tabbar.getId(0);
tabbar.removeCell(id);


~~~
 api_method

<Descr>




</Descr>


 instead of it, you can use the tabbar/api/tabbar_removetab_method.md method:

~~~js
var id = tabbar.getId(0);
tabbar.removeTab(id);
~~~
