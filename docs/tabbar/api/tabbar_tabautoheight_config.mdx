<Short>
 defines whether the height of tabs is automatically adjusted to the height of the tabbar container






</Short>



<Type>
 boolean


</Type>


~~~js
 
var tabbar = new dhx.Tabbar("tabbar_container", {
    views:[
        { tab: "left", css:"panel flex", header:"Left"},
        { tab: "west", css:"panel flex", header:"West"},
        { tab: "east", css:"panel flex", header:"East"},
        { tab: "right", css:"panel flex", header:"Right" }
    ],
    tabAutoHeight: true, /*!*/
    mode: "right"
});


~~~
	api_config

<Descr>

:::note
The property can be applicable when [mode](/tabbar/api/tabbar_mode_config) is set to "right" or "left". 1
:::


</Descr>
