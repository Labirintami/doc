<Short>
 fires when a user enters the value of a control in the input manually
 


</Short>



- inputValue        string  an entered value


~~~js

form.getItem("timepicker").events.on("Input", function(inputValue) {
    console.log("Input", inputValue);
});


~~~
 api_event

<Descr>


:::note
The functionality is available when the "editable" option of the TimePicker control is set to *true*.1
:::


</Descr>


 added in v7.0
