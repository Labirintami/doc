<Short>
 returns an object with the available configuration attributes of the control


</Short>



param   object      an object with the available attributes of the control and their values


~~~js

var form = new dhx.Form("form_container", {
    rows: [
        {
            type: "spacer",
            name: "spacer"
        }
    ]
});

form.getItem("spacer").getProperties();

// -> the returned object
{
	height: "content",
	padding: 0,
	width: "content"
}

~~~
 api_method

<Descr>

The returned object of the <strong>Spacer</strong> control can contain the following configuration attributes:
