<Short>
 allows changing a list of Select options dynamically


</Short>



- options   array   an array with new options



~~~js

form.getItem("select").setOptions([
    { value: "new_1", content: "new_content-1", disabled:true },
    { value: "new_2", content: "new_content-2" }
]);


~~~
 api_method

<Descr>

Each option is an object with a set of *key:value* pairs - attributes of options and
their values:

- <strong>value</strong> - (*string|number*) mandatory, sets the value for the select option
- <strong>content</strong> - (*string*) mandatory, the content displayed in the select option
- <strong>disabled</strong> - (*boolean*) optional, defines whether the option is enabled (*false*) or disabled (*true*)



</Descr>
