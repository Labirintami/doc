<Short>
 sets a new configuration for a form



</Short>



- config	object		a configuration object of a form



~~~js

form.setConfig({
	cellCss: "dhx_widget--bordered",
	gravity: false,
	rows: [
		{
			type: "input",
			label: "Name",
			icon: "dxi-magnify",
			placeholder: "John Doe"
		}
	]
});


~~~
