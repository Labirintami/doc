<Short>
 returns the dhtmlxCalendar widget attached to a DatePicker control




</Short>



param   object    the dhtmlxCalendar widget

~~~js

var datepicker = form.getItem("datepicker").getWidget();  
// -> dhtmlxCalendar


~~~
 api_method

<Descr>

It is possible to use methods of dhtmlxCalendar via the <strong>getWidget()</strong> method of a DatePicker control.

For example, you can show the current month in the control. To do this, you need to get the widget attached to the DatePicker control and then use the [showDate()](/calendar/api/calendar_showdate_method) method of this widget.

~~~js
var datepicker = form.getItem("datepicker").getWidget();  // -> dhtmlxCalendar
datepicker.showDate(null,"month"); // shows the current month
~~~
