<Short>
 checks whether a form is disabled




</Short>



- param	boolean     true, if a form is disabled, otherwise false


~~~js

form.isDisabled(); // -> true/false


~~~
 api_method

<Descr>

To check whether a form control is disabled, pass either the name of the control or its id (if the name attribute is not defined in the config of the control) as a parameter to the <strong>isDisabled()</strong> method:

~~~js
form.isDisabled("combo"); // -> true/false
~~~


</Descr>
