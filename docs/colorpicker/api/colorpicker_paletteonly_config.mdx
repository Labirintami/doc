<Short>
 
defines whether ColorPicker is shown only in the palette mode



</Short>


<strong>Default value: </strong> 
false



<Type>
 boolean


</Type>


~~~js
 
var colorpicker = new dhx.Colorpicker("colorpicker_container",{
	paletteOnly: true
});


~~~
	api_config

<Descr>
 
:::note
Note that in the <strong>paletteOnly</strong> mode the palette is displayed without the panel with custom (user-defined) colors.1
:::


</Descr>
