<Short>
 sets focus on the specified value



</Short>



- value		string		the code of a color (in the Hex format) to set focus on



~~~js

colorpicker.focusValue("#BDF0E9");


~~~

<Descr>



</Descr>

~~~js
colorpicker.setFocus("#BDF0E9");
~~~
