<Short>
 fires on changing the mode of the colorpicker


</Short>



- mode	string		the colorpicker mode ("palette", "picker")


~~~js

colorpicker.events.on("ModeChange", function(mode) {
   console.log("Selected mode: " + mode); 
});

 
 
~~~
